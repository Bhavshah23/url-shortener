# URL Shortener

Simple full-stack URL shortener.

**Live:** https://url-shortener-zeta-gilt.vercel.app/

## Stack
React, Node.js, Express, MongoDB

## Features
- Create short links  
- Custom short codes  
- Redirect to original URL  
- Track clicks  
- Delete links  

## Run locally
```bash
git clone https://github.com/Bhavshah23/url-shortener
cd url-shortener

cd server && npm install && npm start
cd ../client && npm install && npm start
